# checkup-skills

Claude Code용 스킬 플러그인 마켓플레이스입니다. 초보자용 시그니처 스킬(설치·한국어 글쓰기·엑셀·설치 검증·발표자료·회의록·네이버 SEO)부터 프롬프트 작성·한국 공공데이터 API 연동·repo 정리의 실전 검증된 노하우까지 스킬로 묶어 한 번에 설치할 수 있게 했습니다. [claude-checkup](https://github.com/x77xdavid-prog) 프로젝트의 일부로, "설치 장벽 없이 바로 써보는" 무료 스킬 모음입니다.

## 설치

```
/plugin marketplace add x77xdavid-prog/checkup-skills
/plugin install prompt-master@checkup-skills
/plugin install token-diet@checkup-skills
```

두 번째 줄의 `prompt-master`를 아래 표의 다른 플러그인 이름으로 바꾸면 해당 스킬을 설치합니다.

## 시그니처 스킬 (초보자용)

AI 초보자·비개발자를 위해 만든 이 마켓의 대표 스킬입니다. 공통 원칙: 한 번에 한 단계씩 진행하고, 검증을 통과하기 전에는 "완료"라고 선언하지 않습니다.

| 플러그인 | 하는 일 | 트리거 예시 |
|---|---|---|
| **first-setup** | Claude Code를 처음 설치하는 사람을 설치부터 첫 실행 성공까지 대화로 완주시킵니다. 단계마다 확인 방법과 실패 시 처방을 제공합니다. | "클로드 코드 설치하고 싶어", "설치가 안 돼요" |
| **ko-writer** | 블로그·업무 보고서·이메일 등 자연스러운 한국어 글을 번역투·AI 상투어 없이 작성합니다. 짧은 초안으로 방향을 확인한 뒤 전체를 씁니다. | "블로그 글 써줘", "보고서 초안 잡아줘" |
| **excel-helper** | 엑셀·CSV 표 정리, 수식, 요약, 차트를 비개발자 언어로 돕습니다. 원본은 절대 덮어쓰지 않고 사본에서 작업하며, 변경 전후 대조표로 검증합니다. | "이 엑셀 정리해줘", "월별로 집계해줘" |
| **did-it-work** | 스킬·플러그인·MCP 설치 후 "됐는지 모르겠다"를 증거 기반으로 판정합니다. 목록 확인 + 실제 호출 테스트를 통과해야 성공으로 판정합니다. | "설치 됐는지 확인해줘", "이거 작동하는 거야?" |
| **ppt-draft** | 주제만 주면 발표자료(PPT) 초안을 목차→슬라이드 구성→발표 노트까지 만듭니다. 목차 승인 후에만 슬라이드를 채우고, 제목은 명사가 아니라 주장 문장으로 씁니다. | "발표자료 만들어줘", "PPT 초안 잡아줘" |
| **meeting-actions** | 회의록·녹취 텍스트를 결정사항·액션아이템(담당·기한)·미결 이슈로 정리합니다. 원문에 없는 담당·기한은 지어내지 않고 "미지정"으로 표기하며, 모든 액션아이템에 근거 발언을 붙입니다. | "회의록 정리해줘", "액션아이템 뽑아줘" |
| **naver-seo-check** | 네이버 검색 기준으로 블로그 글·웹페이지를 체크리스트 진단하고 우선순위 3개만 처방합니다. 비공개 알고리즘은 [추정] 표기, 순위 보장 발언은 하지 않습니다. | "네이버 상위노출 진단해줘", "내 글 SEO 봐줘" |

## 토큰 절감 (token-diet 패밀리)

Claude Code 세션은 서로 다른 다섯 곳에서 토큰이 샙니다. 한 스킬이 전부를 덮지 못하므로, 상황에 맞는 레버를 골라 쓰거나 지휘 스킬(`token-diet`)에 맡깁니다.

| 플러그인 | 하는 일 | 트리거 예시 |
|---|---|---|
| **token-diet** | 지휘 스킬 — 상황에 맞는 절감 레버(입력·툴콜·출력·코드·예산)를 골라줍니다. | "토큰 줄여줘", "싸게 해줘", "효율적으로" |
| **lean-context** | 입력·컨텍스트 토큰 절감 — 통째로 읽지 말고 위치를 찾아 필요한 줄만, 큰 건 서브에이전트 요약에 위임합니다. | "토큰 아껴", "컨텍스트 꽉 참", "통째로 읽지 마" |
| **quiet-tools** | 툴콜 낭비 제거 — 재읽기·재확인 금지, 독립작업 병렬 배치, 나레이션 제거로 툴콜 오버헤드를 줄입니다. | "재확인 그만", "빨리", "나레이션 하지 마" |
| **no-restate** | 반복 출력 제거 — 서두·맺음말·기확립 사실 재유도를 없앱니다(caveman보다 가벼운 출력 다이어트). | "본론만", "요약 반복 그만", "인사 빼" |
| **token-budget** | 작업에 토큰 목표를 걸고 그 안에서 깊이를 조절하며 소비를 보고합니다. | "50k 안에서", "빠르게 한 번", "토큰 예산" |
